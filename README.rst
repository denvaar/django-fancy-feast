==================
django-fancy-feast
==================

A collection of additional form fields for Django.

Requirements
------------
- python3
- django 1.8 or higher

Quick Start
-----------

1. Run ``pip install django-fancy-feast``

2. Add 'fancy_feast' to your INSTALLED_APPS setting::

    INSTALLED_APPS = [
        ...

        'fancy_feast',
    ]

Examples
--------

Just a couple things so far.
